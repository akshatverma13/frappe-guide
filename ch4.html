Frappe Desk

What coordinators, supervisors, and admins see and do in the browser. List View, Form View, approvals, timeline.

What Is Frappe Desk?

The browser-based back office. No installation. Open a browser, go to the URL, log in. Works on Chrome, Firefox, Edge, Safari.

⚙️
Administrators
  • Create/modify form definitions (DocTypes)
  • Create user accounts and assign roles
  • Configure which forms appear on mobile
  • Add or modify dropdown option lists
👥
Coordinators / Supervisors
  • Review data submitted by field workers
  • Approve or reject submissions
  • Run reports and export data
  • Manage master data (geography, employees)
