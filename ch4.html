What Is Frappe Desk?

The browser-based back office. No installation. Open a browser, go to the site URL, log in. Works on Chrome, Firefox, Edge, and Safari.

⚙️
Administrators
  • Create and modify form definitions (DocTypes)
  • Create user accounts and assign roles
  • Configure which forms appear on mobile
  • Add or modify dropdown option lists
👥
Coordinators / Supervisors
  • Review data submitted by field workers
  • Approve or reject submissions
  • Run reports and export data to CSV/Excel
  • Manage master data (geography, employees)
