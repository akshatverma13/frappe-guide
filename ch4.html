Frappe Desk

The browser-based back office. List View, Form View, Report Builder, Dashboards, Workspaces, and the full approval system.

What Is Frappe Desk?

Frappe Desk is the browser-based interface that comes built into every Frappe installation. It is a full-featured back office — no installation needed, works on any modern browser. It is the primary interface for coordinators, supervisors, and administrators.

⚙️
Administrators
  • Create/modify DocTypes and fields
  • Create user accounts, assign roles
  • Configure mobile app settings
  • Manage dropdown option lists
  • Set up workflows and assignment rules
👥
Coordinators / Supervisors
  • Review and approve submitted records
  • Run and export reports
  • Manage master data (employees, geography)
  • Add comments and corrections to records
  • Monitor dashboards and KPIs

Navigating Desk

📁Sidebar — Modules and DocTypes▼

The left sidebar lists all Modules. Click a module to expand it and see all its DocTypes. Click any DocType to open its List View. You can pin frequently used items to the top of the sidebar.

🏠Workspaces — Customizable Dashboards▼

A Workspace is a customizable landing page for a module. It can contain shortcuts, summary numbers, charts, links to reports, and notes. Each module typically has its own Workspace. Users can create personal Workspaces.

🔍Global Search — Find Anything▼

The search bar at the top (keyboard shortcut: Ctrl/Cmd + G) searches across all DocTypes, records, and settings. Start typing a record name, DocType, user, or setting and it appears instantly.

🔔Notifications and Assignments▼

The bell icon shows all notifications — records assigned to you, mentions in comments, workflow transitions. The "Assigned to Me" view (⊹ icon) shows all your pending tasks across all DocTypes.

Form View — One Record in Full

📄
Sections as Cards

Fields grouped into visual cards. Each Section Break creates one card. Can be collapsed.

📑
Tabs at the Top

Long forms use Tab Breaks to create navigable sections — click to switch between tabs.

📊
Live Child Tables

Add, edit, reorder, delete rows in child tables directly in the form view.

👁️
Live Conditions

Conditional fields show/hide in real-time as values change — identical to mobile behavior.

⚠️
Inline Validation

Validation errors appear next to the problem field — not as pop-ups.

📜
Sidebar Links

Related records (other DocTypes linked to this one) appear in the right sidebar for quick navigation.

Report Builder

Frappe has four types of reports, each suited to different needs:

Report TypeHow It's BuiltBest For
Report BuilderVisual drag-and-drop — no codeSimple column-based tabular reports, quick filtering
Query ReportSQL query written by developerComplex joins, aggregations, custom calculations
Script ReportPython code for data + optional JS for columnsDynamic reports with complex business logic
Dashboard ChartConfiguration UI — no codeVisual charts: bar, line, pie, doughnut, heatmap
